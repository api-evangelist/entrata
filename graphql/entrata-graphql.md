# Entrata GraphQL API

Entrata is a property management platform for multifamily housing. The API covers property listings, leasing applications, resident management, rent collection, maintenance requests, vendor management, and operational analytics.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Entrata REST API
**Documentation:** https://docs.entrata.com/api/v1/documentation

- Reference: https://developer.entrata.com/catalog/all
- Schema: https://github.com/api-evangelist/entrata
